Componenti principali: Stage, Scene, Scene Graph - Nodes

**************** STAGE ******************
uno Stage è la finestra che contiene gli oggetti dell'applicazione JAVA FX.

utilizza la classe Stage,passata come argomento al metodo start() impostando Width e Height

è diviso in Content Area e Decorations

ci sono 5 tipi di stage:
- Decorated
- Undecorated
- Transparent
- Unified
- Utility

per mostrare il contentuto di uno stage, si chiama il metodo show()


*************** SCENE ********************
rappresenta i contenuti "fisici" dell'applicazione

utilizza la classe Scene

si istanzia passando height e width, e volendo un root node al costruttore


*************** SCENE GRAPH - NODES ********************
uno scene graph è una struttura ad albero che rappresenta gerarchicamente i contenuti di una scena

un nodo è un oggetto grafico di uno scene graph

un nodo può includere:
- Geometrical (Graphical) objects (2D and 3D) such as − Circle, Rectangle, Polygon, etc.
- UI Controls such as − Button, Checkbox, Choice Box, Text Area, etc.
- Containers (Layout Panes) such as Border Pane, Grid Pane, Flow Pane, etc.
- Media elements such as Audio, Video and Image Objects.

un nodo è rappresentato dalla classe Node.

possono essere di tre tipi:
- Root Node: è il primo scene graph, obbligatorio passarlo
- Parent Node: è un nodo con dei figli. la classe base è la classe astratta Parent che può essere
  dei tipi:
  	- Group: un nodo gruppo contiene una lista di figli. quando il gruppo è renderizzato,
  	  tutti i figli sono renderizzati in ordine. ogni effetto applicato al gruppo, è applicato ai figli
  	- Region: è la base class per i controlli della UI come Chart,Pane,Control
- Leaf Node: è un nodo senza figli. (Box,ImageView,Mediaview...)

******************* LIFECYCLE *************************
start() − The entry point method where the JavaFX graphics code is to be written.

stop() − An empty method which can be overridden, here you can write the logic to stop the application.

init() − An empty method which can be overridden, but you cannot create stage or scene in this method.
